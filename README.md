# Predicting Hospital Readmission Risk Using Machine Learning

## Project Overview
Hospital readmissions are one of the major challenges faced by healthcare organizations. Frequent readmissions increase operational costs, reduce bed availability, decrease patient satisfaction, and may lead to financial penalties from insurance providers.
This capstone project aims to build a machine learning model that predicts whether a patient is likely to be readmitted using historical healthcare data.

## Business Problem
MediCare Health Network wants to identify high-risk patients before discharge so that appropriate interventions can be planned to reduce avoidable readmissions.

## Objectives
- Understand patient demographics and clinical characteristics
- Perform Exploratory Data Analysis (EDA)
- Build predictive classification models
- Compare multiple machine learning algorithms
- Recommend interventions for high-risk patients
